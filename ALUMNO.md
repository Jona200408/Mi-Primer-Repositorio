Que es Git

Git es una herramienta que sirve para guardar los cambios que haces en un proyecto. Es como un historial donde se va guardando todo lo que modificas. Si te equivocas o borras algo, puedes regresar a una version anterior sin problema. Tambien sirve mucho cuando trabajas en equipo porque no se mezcla el trabajo de todos.

Por que necesitamos Git

Git es importante porque sin el es muy facil perder archivos o confundirse con tantas versiones. A veces uno guarda archivos con nombres diferentes y luego ya no sabe cual es el mas nuevo. Con Git todo queda ordenado y puedes ver los cambios que hiciste. Tambien puedes subir tu proyecto a GitHub para tenerlo guardado en internet y no perderlo.

Los 3 conceptos clave de Git

Repositorio
Es la carpeta donde esta tu proyecto y donde Git guarda todo el historial de cambios. Es como el lugar principal donde esta todo el trabajo.

Commit
Es cuando guardas los cambios que hiciste. Cada commit es como guardar el progreso de tu proyecto y puedes poner un mensaje para recordar que hiciste.

Rama
Es como una copia del proyecto donde puedes hacer cambios sin afectar el principal. Sirve para probar cosas nuevas sin arruinar el proyecto original.

Los comandos que aprendi

git init
Sirve para empezar Git en una carpeta y convertirla en repositorio

git add
Sirve para seleccionar los archivos que quieres guardar

git commit
Sirve para guardar los cambios con un mensaje

git push
Sirve para subir los cambios a GitHub

git pull
Sirve para bajar los cambios mas recientes del repositorio
